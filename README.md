# Hyprland window switcher

This is a plugin for the Albert launcher to switch between Hyprland windows.

## Installation

```bash
cp -r hyprland_window_switcher ~/.local/share/albert/python/plugins/
```
Or clone the repo directly there;

## Missing features

App icons
A lot of other hyprland related commands (with hyprctl), could be fun to add

## Usage

Default: `hypr <window-name>`
